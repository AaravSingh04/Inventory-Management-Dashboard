# 📦 Inventory Management Dashboard

An interactive Inventory Management Dashboard built with **Python**, **Streamlit**, **Pandas**, and **Plotly**. It helps visualize stock levels, identify low-stock items, analyze inventory by category and supplier, and calculate inventory valuation.

## 🚀 Live Demo

https://inventory-management-dashboard-ewvcqfqph8zqmxczzc6fal.streamlit.app/

## 🛠️ Tech Stack

- Python
- Streamlit
- Pandas
- Plotly

## ✨ Features

- 📦 Stock Levels
- ⚠️ Low Stock Alerts
- 📊 Category Analysis
- 🚚 Supplier Analysis
- 💰 Inventory Valuation
- 📈 Interactive Plotly Charts

## ▶️ Run Locally

1. Clone the repository:

```bash
git clone https://github.com/AaravSingh04/Inventory-Management-Dashboard.git
```

2. Navigate to the project folder:

```bash
cd Inventory-Management-Dashboard
```

3. Create a virtual environment:

```bash
python -m venv venv
```

4. Activate the virtual environment:

**Windows**
```bash
venv\Scripts\activate
```

5. Install dependencies:

```bash
pip install -r requirements.txt
```

6. Run the application:

```bash
streamlit run app.py
```
